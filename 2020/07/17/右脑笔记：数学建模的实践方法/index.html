右脑笔记：数学建模的实践方法

发表于 2020-07-17 分类于 数学建模
问题的识题与分析
  1. 建模的目的是什么？
  2. 需要解决的问题是什么？核心问题是什么？
  3. 问题的关键词是什么？
  4. 解决问题的思路是什么？
  5. 问题有什么特点、限制条件？
  6. 从条件和数据可以得到什么信息？
  7. 数据的来源是否可靠？
  8. 所给条件和数据有什么意义和作用？
  9. 哪些条件是本质的？
  10. 哪些条件中可变动？
  11. 是否需要适当补充一些条件和数据？
数学模型的建立方法
给出合理假设
  1. 关于是否关于包含某些因素的假设
  2. 关于条件相对强弱的假设
  3. 关于各因素影响相对大小的假设
  4. 关于模型适用范围的假设
模型的求解方法
  1. 尽量使用自己熟悉的方法
  2. 用解析方法或数值方法
  3. 使用工具软件说明使用原理和程序，避免黑箱操作
  4. 注意问题的针对性和适用性
模型解的分析和检验方法

误差分析：

  1. 建模假设产生的误差
  2. 近似求解方法产生的误差
  3. 计算机产生的舍入误差
  4. 测量或实验数据产生的误差

模型结果分析：

  1. 分析求解结果是否满足实际问题的需求
  2. 如有偏差，分析其原因，假设或模型，还是求解方法？
模型解的分析和检验方法
灵敏度分析：
  1. 某些参数的扰动对模型结果产生的影响
  2. 从工程研究的角度必须要做的工作
模型的模拟检验或仿真
  1. 模拟与仿真的内涵
  2. 数学建模工程问题是必须要做的工作
  3. 利用已有的模型进行模拟检验
  4. 利用已有的算法思想进行模拟或仿真
竞赛论文的写作方法

评价论文的主要标准
  1. 假设的合理性
  2. 建模的创造性
  3. 结果的正确性
  4. 表述的清晰性
