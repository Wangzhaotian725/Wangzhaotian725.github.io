右脑笔记：决策分析

发表于 2020-07-25 更新于 2021-03-02 分类于 数学建模

决策分析的基本问题

决策问题的要素

  1. 决策者
  2. 可供选择的方案
  3. 准则：衡量选择方案，包括目的，属性的正确性标准
  4. 事件：不为决策者控制的客观存在的将要发生的状态
  5. 每一事件的发生将会产生某种结果
  6. 决策者的价值观

按决策环境

确定型决策

确定型决策是指自然环境完全确定，做出的选择也是确定的。未来状态已知，只需从备选决策方案中挑选出最优方案。

能够引起决策问题的不确定性的因素，称为自然状态，由于自然状态的不确定性导致后果不确定。

不确定型决策

不确定型决策是决策者对将要发生结果的概率无法确定或者一无所知，只能凭借主观意向进行的决策。

决策者面临的决策环境由一些自然状态组成，决策者可以采取若干决策方案，每一种决策方案在不同自然状态下出现的结果是已知的，但决策者不能预先估计各种自然状态出现的概率。

​ 不确定决策的几种准则：

​ 悲观准则

​ 乐观准则

​ 后悔准则

​ 折中准则

​ 等可能性准则

风险型决策

自然环境不完全确定，但是其发生的概率是可以推算或者已知的。

决策者采取某种行动方案后系统有若干种状态可能发生，决策者对系统的状态虽然没有控制力，但是却能确定每种状态发生的概率。

决策分析的基本步骤：
  1. 构造决策问题，为决策问题提供可能的方案，并标定目标
  2. 确定各种决策可能的后果并设定各种后果发生的概率
  3. 决定各种后果对决策人的实际价值
  4. 进行评价和比较
风险型决策的期望值法

期望值法是根据各可行方案在各自然状态下收益值和概率平均值的大小，决定各方案的取舍。

决策者具有以下假定：

  1. 无差异性：一个确定收益和一个与之等值的期望收益之间无差异
  2. 趋利性：期望收益值越大越好

决策树方法

决策树方法的步骤
  1. 画决策树
  2. 预测事件发生的概率
  3. 计算损益值
信息价值（EVPI）

完全信息：决策者能完全肯定未来那个自然状态会发生

若能获得完全信息，风险型决策便转化为确定型决策，提高决策准确性。要获得信息必须支付一定的费用，决策者应首先估计这些信息的价值。
